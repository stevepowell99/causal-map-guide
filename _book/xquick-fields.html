48 🧪 Quick fields

This feature is experimental and the details will certainly change.

Quick fields can be created with a special kind of flag for factor labels. You can add information about one or more dimensions using a format like this:

Crops improved time=after

Temporary fields will be created in the factors table which you can then use

Quick fields must be composed only of letters and numbers, no punctuation or spaces. There must be no gap on either side of the equals sign.

You can combine more than one quick field at once:

Crops improved time=after valence=2

Quick links can appear anywhere in a label and in particular anywhere within a hierarchy:

Crops improved; time=after valence=2

Crops improved time=after; valence=2

Took part in training theme=gender; online sessions frequency=weekly
