When you are looking at one statement at a time, you will see the navigation bar displayed below.

image-20211105092246054

The info toggle opens additional statement info, i.e.Â info for any fields which start with #, for example you might have a field #gender in your statements table.

sm559T6lUE

Boxes to edit the statement, question and source memos will also appear. Here you can add useful additional data which can be displayed on your map using the label links or label factors filters.

chrome_xXrp2x0TKw

By clicking on the green book icon you can see all the statements from all sources or the black book icon will show the statements from the the respondent who said the statement you are currently viewing.

image-20211108162324112

You can use the arrow and numbered button to skip through statements. If you want to skip to a particular statement the easiest way is to use the find statements filter. This will automatically be applied if you are viewing statements one at a time, so simply click on the filter and edit the number it is searching for.
