💻 Text analysis in the app

From the inline help

Text Analysis

This panel shows analysis on the text you have uploaded to the app. Currently the most frequently used words are ordered and shown - for the entire map, and for the current filter. The analysis shown here is not just on coded quotes but on all the statements which you have uploaded to the app. The 'mean sentiment' is also shown. This is a number indicating how 'positive' or 'negative' a respondent's response was. This is worked out by assigning a number to any word that has a positive/negative connotation in the statement. For example, use of the word 'nice' is +3, 'bad' is -3 etc. This shouldn't be treated as a definite answer to whether or not a statement implied a positive/negative response to the question - but it can give an idea.
