Analysis: viewing and filtering the map

Various kinds of software are available to simplify the task of viewing and filtering a causal map, especially for larger maps. For example, gephi.org is a good choice for very rapidly manipulating even very large maps - much larger than Causal Map can handle.

A causal map can be filtered to show only some parts of it. Some submaps are well-known from network analysis, for example the â€œego networkâ€ for a particular factor is the part of the map which includes only those factors directly connected to it. This is something like searching for that particular factor.

An related approach is to specify one set of Source factors and another set of Target factors and to display only the links along paths which lead from a Source factor to a Target factor. This is called an â€œetiographâ€ in a seminal organisational analysis of Utrecht Jazz Orchestra (Bougon, Weick, & Binkhorst, 1977).

From the inline help

The Interactive Map

This map has many features. You can:

  • Drag the factors around
  • Copy a map as an image by right-clicking on it.
  • Save a PNG image by pressing the button at bottom right.
  • Change the aspect ratio using the slider.
  • Hover over factors to highlight the connected links and factors.
  • Hover over factors to display basic information about them (and “edit” link also appears but we won't be using that yet).
  • Hover over links to see the associated quote in a panel. You can also switch this panel off.
  • Click on links to edit them.

If the map won't settle down or looks wrong, try pressing the blue “refresh” button. Map views with more than 750 factors are not shown.

You can maximise the map to see everything in more detail by pressing the maximise box on the right.

If you have been coding using “strength” and “actualisation” of links, factors are coloured green through white to red to show how much they are affected by “increasing” influences (see Advanced section below). As a result of this, parent-less factors (“drivers”) at the start of chains are always white. The size of factors reflects the number of mentions.

Searching and filtering factors

Type to select factors which you want to find.

  • Select one or more pre-existing factors. Press enter to accept

  • And/or just type fragments of text which might match several factors; type a comma to complete

The map changes:

  • All factors matched by the search are highlighted.

  • When viewing all statements, only the factors matched by the search and those upstream and downstream (left and right) of them are shown, as many steps as set by the Upstream steps and Downstream steps sliders.

Factors are only matched if the search term uses the same case, e.g. “Farming” will not match “farming” or “FARMING”.

If you also enter a search term in the second “Target” box, the app traces all the paths from the factor(s) identified in first box to the factor(s) identified in this “Target” box, up to the maximum number of steps which you specify with the Downstream steps slider.
