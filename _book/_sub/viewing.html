You will be able to:

  • See the overall map
  • Filter the global map by: Current statement; Factor label; Link (quote, hashtag, memo etc); Statement (text, additional data e.g.Â respondent characteristics)
  • See a summary of all statements from a specific source
  • See additional information about each statement (e.g.Â question and respondent characteristics)
  • Simplify the map byÂ â€œzooming outâ€ or showing only most frequent factors or links
  • Trace paths from one set of factors to another

  • Create detailed tables: Factors; Sample; Questions; Differences; Closed questions

Start your journey at go.causalmap.app. You will see a screen like this:

You can log into the app using the email and password you have been given. (Note: the email field is not case sensitive)

You will then see a screen like this:

The files you are able to view, are listed in the first box in the file chooser (above), with the ones you viewed most recently listed first. You can just use your mouse to select a file, or, as with all menus like this in Causal Map, you can press your delete key and type to search for files.

You should then wait for the file to fully load (if it takes longer than a minute, try pressing â€œDismissâ€).

You will spend nearly all of your time in the Code & View tab, which is described further down in this section.

ğŸ’» The left-hand panels

You can resize the left-hand panels in relation to the right-hand panels by dragging the right-hand edge of the left panel to the left or right.

Filters

There are various filters which can be applied to the maps, quotes, and tables to analyse the data.

More information on the individual filters can be found later on in this Guide.

Firstly, if you want to view the dataset as a whole, make sure View statements one by one filter is off by clicking on the toggle (it should turn from green to grey).

image-20210108212920731

The Show filters panel should then automatically open. If it doesnâ€™t, click on the toggle (it should turn from grey to green).

image-20210108212929350

  • You can open and close each filter shown in the filter panel by using the toggles.

  • When you select and apply a filter, the visualisations, quotes, and tables on the right will update.

  • You can apply multiple filters at once.

  • The filter bar will be highlighted in pink if you have applied any filters â€“ always be aware of which filters you have applied and remember to be transparent about these in your reporting.

From the inline help

The left panels

All the panels on the left (Search and Filter Factors, Simplify, Search and Filter Statements, Search and Filter Links and the pager “View Statements one by one”):

  • Can all be used separately, e.g. you can Search and Trace and Zoom at the same time.
  • Can be opened or closed.
  • Become inoperative when you close them.


The buttons in the lower half let you switch between coding and editing links, or editing the names of your factors.

Viewing your statements

From the inline help

Viewing statements one by one or all together

When View statements one by one is selected:

  • The map on the right is filtered to show only codings made in the current statement.
  • You can “page” through the statements using the buttons to go to the first, the previous, the next and the last statements, or use the dropdown menu in the middle to go directly to any statement if you know its number (press your delete key to clear the number first).

  • When you move to another statement, the corresponding text appears below.

  • TIP: Press the >>| button to go to the last statement you have coded.

When it is not selected:

  • The map shows all codings from all statements (unless you are using other filters).

ğŸ’» The right-hand panels

There are four main panels which present and visualise the data. More information on each can be found later on in this Guide.

Interactive

This interactive causal map displays all the causal links between factors which correspond with your search/filter criteria. You can drag factors around using your mouse to rearrange the map. You can also view the quotes associated any of the causal links by hovering over the arrow that connects the two factors.

Print

Here youâ€™ll find a high-quality static map which is particularly useful for including in reports as it presents the causal stories clearly.

Quotes

Here you will see a list of all the quotes which correspond to the filters you have applied. This is useful for selecting and copying quotes related to particular stories of change, to include in the report.

Quantity of Evidence (QE)

If you have selected factors in both the â€œSource Factorâ€ and â€œTarget Factorâ€ filters, the app traces all the paths from the Source factor(s) to the Target factor(s). This panel then calculates for you the robustness of the argument that the source factor(s) influence the target factor(s).

Tables

Here you will find a selection of tables giving information about the factors and links in the current view, and other data associated with them, such as characteristics of the corresponding sample of respondents.
